## Исследование надежности заемщиков

### Описание проекта
В проекте требуется исследовать данные о платёжеспособности клиентов банка, а затем ответить на вопросы:

1. Есть ли зависимость между количеством детей и возвратом кредита в срок?
2. Есть ли зависимость между семейным положением и возвратом кредита в срок?
3. Есть ли зависимость между уровнем дохода и возвратом кредита в срок?
4. Как разные цели кредита влияют на его возврат в срок?

Результаты исследования будут учтены при построении модели кредитного скоринга — специальной системы, которая оценивает способность потенциального заёмщика вернуть кредит банку.

### Результат работы по проекту
1. Проведена предобработка данных: заполнены пропуски, исправлены аномалии, изменены типы данных, удалены дубликаты. Выполнена декомпозиция исходной таблицы (данные об образовании и семейном статусе вынесены в отдельные таблицы). Произведена категоризация дохода клиентов и целей кредита.
2. Применены различные виды группировки данных. Даны ответы на сформулированные в проекте вопросы. Поставленная в проекте задача выполнена.
